# The Grand Cru Atlas

An interactive map of the world's greatest vineyard sites, organized by grape variety. Explore ~200 vineyard locations across 16 grape categories — from the blue Devonian slate of the Mosel to the galets roulés of Châteauneuf-du-Pape — with terroir descriptions and key producers for each site.

## Live Site

**[→ Open the Grand Cru Atlas](https://jskarabot18.github.io/grand-cru-atlas/)**

## What's Inside

- **Interactive map** built with [Leaflet.js](https://leafletjs.com/) on a clean CARTO basemap
- **~200 vineyard sites** across 16 grape categories: Riesling, Chardonnay, Pinot Noir, Bordeaux Blends, Nebbiolo, Sangiovese, Malbec, Syrah/Shiraz, Cabernet Franc, Rhône Blends, Cabernet Sauvignon, Grenache, Tempranillo, Chenin Blanc, Grüner Veltliner, and Gamay
- **For each site:** terroir description, soil/geology, key producers
- **Filtering** by grape category, search by vineyard name, region, or producer
- **Downloadable PDF** of the full atlas

## How to Use

Open the [live site](https://jskarabot18.github.io/grand-cru-atlas/) in any modern browser. Click a grape category in the left panel to filter the map, click any pin for vineyard details, or use the search bar to find a specific vineyard, region, or producer. Press `Esc` to reset the view.

To run locally, simply open `index.html` in a browser — no build tools or server required.

## About

This project grew out of 20+ years of wine collecting and study. It is a personal reference tool mapping the vineyard sites I consider the most important in the world, organized not by appellation hierarchy but by grape variety and terroir.
